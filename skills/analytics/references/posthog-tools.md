# PostHog MCP — Справочник инструментов

URL: `posthog.larixon.com` (self-hosted)
Подключение: через локальный прокси на `localhost:8787` (запускается автоматически)

## Доступные возможности

PostHog MCP предоставляет доступ к API PostHog через MCP-протокол. Основные области:

### Аналитика и тренды
- Получение трендов по событиям (`$pageview`, `$screen`, кастомные)
- Фильтрация по датам, свойствам, проектам
- Разбивка по дням/неделям/месяцам

### Воронки (Funnels)
- Построение воронок из последовательности событий
- Расчёт конверсии между шагами
- Фильтрация по свойствам пользователей

### Когорты (Cohorts)
- Просмотр существующих когорт
- Создание когорт по условиям

### Фичфлаги (Feature Flags)
- Список активных флагов
- Включение/выключение флагов
- Создание новых флагов

### Эксперименты (Experiments)
- Список A/B экспериментов
- Создание экспериментов
- Просмотр результатов

### Дашборды
- Список дашбордов
- Создание новых дашбордов

### Аннотации
- Добавление аннотаций к графикам (например: «релиз v2.5»)

### Ошибки (Error Tracking)
- Просмотр ошибок за период
- Группировка по типу

## Проекты

На нашем PostHog несколько проектов. При запросах можно указать проект по имени или ID.

## Типичные запросы

```
Покажи тренд по $pageview за последние 30 дней
Какие фичфлаги сейчас активны?
Создай A/B эксперимент для нового чекаута
Покажи когорты пользователей
Какие ошибки были за последние 24 часа?
Построй воронку: регистрация → первый проект → деплой
Добавь аннотацию: релиз v2.5 вышел сегодня
```

## Как работает подключение

PostHog MCP использует **локальный прокси** (Cloudflare Worker) на `localhost:8787`, который перенаправляет запросы к `posthog.larixon.com`. Прокси **запускается автоматически** при старте Claude Code.

При первом запуске скрипт клонирует `posthog-mcp` и устанавливает зависимости — это может занять 1-2 минуты. Последующие запуски — мгновенные.

Если PostHog недоступен, для ручной диагностики:
```bash
cd ~/posthog-mcp/typescript
POSTHOG_BASE_URL=https://posthog.larixon.com npx wrangler dev
```
